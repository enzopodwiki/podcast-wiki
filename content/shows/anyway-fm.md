中国大陆 **视觉/UI 设计垂类** 的代表性独立播客，2015 年由两位资深 UI 设计师 **JJ** 与 **Leon** 业余创办，双周更新。从 UI 设计切入，逐步扩展到数码科技、生活美学，核心始终围绕"视觉设计"。开播五期即入选 Apple Best of 2015 最佳播客。属 [[design-podcast|设计播客]] 垂类。来源：[[roles/liqi|利器]]「利器 x 播客」访谈。

## 双主播

- **JJ（JJYing）**：从业十余年的 UI 设计师，早年写[设计博客]，自学音频剪辑与前后端编程；负责剪辑、发布与官网开发。
- **Leon（leongao）**：同为 UI 设计师，曾任职 frog；负责外联、社交账户维护、文字与招贴插画/纪念品设计。
- 二人无明确分工，选题共同构思。JJ 太太生产期间由 Leon 承担较多剪辑。

## 制作与"土味硬核"的技术选型

- **自建 host + 自建官网**：不愿用国内托管服务（插广告），又担心 Fireside.fm 等国外服务在国内的访问速度，于是自己托管音频文件、用老牌 CMS Bo-blog 二次开发搭官网；JJ 自写 PHP 小工具 **Podcast RSS Editor**（开源）管理单集与统计。
- **录音硬件**：舒尔 MV51 ×2（室内）、Sony D100（外出采访）；租了间安静的屋子做 Anyway.HQ。JJ 认为"安静、合适、回音小的房间"比设备更重要。
- **软件**：Trello（选题看板）、Adobe Audition（JJ 全程线性波形剪辑；Leon 偏好 Multi-Tracks 非线性）、Bear（发布 checklist + shownotes）。
- **MP3 章节标记**走在国内前列：建台初期即手动写章节，后用 Podcast Chapters App 适配主流播客 App 的章节功能。

## 商业化与会员

- **会员计划 Anyway.Member**（2017 年底推出）：微信会员群、加长版音/视频、线下活动票、304 钢质会员铭牌、纪念品、官网 X 轴时间轴播放器。
- **免费邮件通讯 Anyway.Post**（安妮薇邮报）：3000+ 订阅，间隔周触达听众——故会员计划不含会员通讯（与多数友台相反）。
- 已买下 anyway.tv 域名，暗示视频化意图。

## 对播客前景的判断（2019）

二人对国内播客发展偏 **悲观**：作为载体"时效不如直播、内容丰富度不如视频、信息检索便捷性不如文字"，总体小众；"广告"在播客里很难转化，靠此变现难以可持续。详见 [[trends/podcast-ads|播客广告]]、[[markets/china-mainland|中国大陆市场]]。

## 同圈与启发

- 启发源：《[[design-podcast|字谈字畅]]》（TheType / TIB）——"连字体排印这么小众的话题都能做播客，我们为啥不行"。
- 常听/对标：机核 Gadio（视为努力目标）、[[story-fm|故事 FM]]（寇爱哲谈录音"人离麦近"）、UX Coffee、原来是这样、科技聚变 TechFusion。
